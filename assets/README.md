# Portfolio Assets

This folder contains images and other assets for your portfolio website.

## Required Images

Replace these placeholder files with your actual images:

### Profile Image
- **File**: `profile.jpg`
- **Recommended size**: 400x400px (square)
- **Description**: Your professional headshot or profile picture
- **Format**: JPG or PNG

### Project Images
- **Files**: `project1.jpg`, `project2.jpg`, `project3.jpg`, `project4.jpg`
- **Recommended size**: 800x600px or 16:9 aspect ratio
- **Description**: Screenshots or mockups of your projects
- **Format**: JPG or PNG

### Resume
- **File**: `resume.pdf`
- **Description**: Your resume/CV in PDF format

## Image Guidelines

1. **Optimize your images** before uploading:
   - Use tools like TinyPNG, ImageOptim, or Squoosh
   - Target file size: < 500KB for best performance

2. **Use descriptive alt text** in your HTML for accessibility

3. **Consider using a CDN** for faster loading times (e.g., Cloudinary, imgix)

## Placeholder Images

Until you add your own images, you can use placeholder services:
- https://via.placeholder.com/
- https://unsplash.com/ (free stock photos)
- https://picsum.photos/ (random placeholder images)

## Adding New Images

1. Place your image files in this `assets` folder
2. Update the `src` attributes in `index.html` to match your filenames
3. Ensure file names match exactly (case-sensitive on some systems)
